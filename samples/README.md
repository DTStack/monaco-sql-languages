## Dev: cheat sheet
### Pick a sample
e.g. esm-plain-webpack
```bash
cd esm-plain-webpack
```

### Install dependencies
```bash
npm install
```

### Open the dev web
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview the build artifact
Preview is powered by [cup](https://github.com/wewoor/cup).
```bash
npm run build
```

```bash
npm run preview
```

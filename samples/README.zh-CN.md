## 开发备忘录
### 选择一个示例
e.g. esm-plain-webpack
```bash
cd samples/esm-plain-webpack
```

### 安装依赖
```bash
npm install
```

### 以开发模式启动
```bash
npm run dev
```

### 构建
```bash
npm run build
```

### 预览构建产物
预览构建产物功能由 [cup](https://github.com/wewoor/cup) 提供支持。
```bash
npm run build
```

```bash
npm run preview
```
